Project Overview

Analysing Children in Low-Income Households Across London, 2014–2021

This project analyses children living in low-income households across London boroughs between 2014 and 2021. The dataset contains ward-level records grouped by borough, allowing borough-level patterns and year-on-year distribution changes to be explored.

The analysis focuses on descriptive statistics, borough-level comparison, visualisation of yearly distributions, and statistical testing to assess whether low-income household figures changed significantly between 2014 and 2021.

Data Dictionary

Wards Sub divisions of boroughs

Filtering Unusual Boroughs

The analysis addresses four key questions:

  1. Calculate mean, standard deviation, minimum and maximum values for each London borough.
  2. Exclude boroughs with unusually small counts to improve comparability.
  3. Visualise yearly distribution patterns with mean and standard deviation.
  4. Compare 2014 and 2021 using a Welch two-sample t-test.

Statistical Comparison: 2014 vs 2021

# Filter data for the earliest year (2014)
data_2014 <- data %>% filter(year == 2014)

# Filter data for the latest year (2021)
data_2021 <- data %>% filter(year == 2021)
# Perform a two-sample t-test

t.test(data_2014$children, data_2021$children, alternative = "two.sided")
## 
##  Welch Two Sample t-test
## 
## data:  data_2014$children and data_2021$children
## t = -4.0215, df = 1095.2, p-value = 6.18e-05
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  -112.4091  -38.6873
## sample estimates:
## mean of x mean of y 
##  528.3214  603.8696
t_test_result <- t.test(data_2014$children, data_2021$children, alternative = "two.sided")

After applying the t-test to the two sets of data about the number of children living in low-income housing in London in 2014 and 2021, the test statistics yielded a value of -4.0215, indicating a difference in the available data’s mean.

Null Hypothesis Significance Testing (NHST):

The NHST method determines whether an event is statistically significant or a random occurrence. Where H0 is the null hypothesis, stating no difference in statistical data, and Ha is the alternative hypothesis, contradicting the null hypothesis. To arrive at a solution, this method takes into account the computed p-value and t-value from the t-tests.

# Report the results using Null Hypothesis Significance Testing (NHST)
#Display the Null Hypothesis
cat("Null Hypothesis Significance Testing (NHST):\n")
## Null Hypothesis Significance Testing (NHST):
#Specifying the null hypothesis H0 and Ha to be an alternative hypothesis
cat("H0: The means of children in low-income households in 2014 and 2021 are equal.\n")
## H0: The means of children in low-income households in 2014 and 2021 are equal.
cat("Ha: The means of children in low-income households in 2014 and 2021 are not equal.\n")
## Ha: The means of children in low-income households in 2014 and 2021 are not equal.
# printing the statistical value 
cat("t-value:", t_test_result$statistic, "\n")
## t-value: -4.021485
#printing the probability value
cat("p-value:", t_test_result$p.value, "\n")
## p-value: 6.179676e-05

Using the Estimation approach

Insights and Interpretation

This analysis shows how the number of children living in low-income households varied across London boroughs between 2014 and 2021. Boroughs with unusually small counts were excluded to ensure a more meaningful comparison across the remaining boroughs.
